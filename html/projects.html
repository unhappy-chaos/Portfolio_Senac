    Projeto do uso de API

    Através do uso de JavaScript e DOM em uma página dinâmica, transformei dados recebidos de uma API que consulta o clima em um layout refinado, atraente que usa de tratamento de erros para impedir repetições. Organização completa do site a cada bloco de divisão e projeto feito sozinho.

    Aluratube

    Esse projeto foi desenvolvido ao longo do curso fornecido pela Alura. Ele simula filtros, usa significativamente uma API, tem seu código totalmente organizado graças a mim e utiliza do Node.js para o server dos dados que são recebidos e devolvidos no formato de JSON.

    Web projeto

    Ao longo do que se foi feito e aprendido nesse projeto conforme os ensinamentos e práticas da Alura, fui capaz de produzir um input para guardar os itens do úsuario dentro da mochila, e fazer com que permanecessem armazenados através do próprio navegador.
